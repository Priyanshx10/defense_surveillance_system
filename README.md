# Defense Surveillance System (DSS)

A modular, real-time video surveillance system inspired by defense and critical infrastructure security platforms.

## Current Capabilities

- Robust video ingestion pipeline
- Frame-by-frame processing
- Performance (FPS) monitoring
- Graceful shutdown and resource management

## Architecture (Current Phase)

Video Source → Ingestion Module → Display

## Tech Stack

- Python
- OpenCV
- NumPy

## Status

Phase 1: Video Ingestion

This project is for educational and demonstration purposes only.
